Question DSE25PII_Q09
  1. Let p(x)=nx33nx+36, where n is a constant. If x+3 is a factor of p(x), then p(3)=
    1. 2.
    2. 0.
    3. 2.
    4. 72.

Solution

We are given p(x)=nx33nx+36, where n is a constant, and x+3 is a factor of p(x). We are asked to find p(3).


Step 1: Use the Factor Theorem to find n.

Concept Note: The Factor Theorem
Remember the rule: If a polynomial p(x) has a factor (xk), then p(k)=0.
Here, the factor is (x+3). We can rewrite this as (x(3)), so our k is 3. Therefore, we must set p(3)=0.

Since x+3 is a factor of p(x), by the Factor Theorem we have p(3)=0. Substitute x=3 into p(x): p(3)=n(3)33n(3)+36=n(27)+9n+36=27n+9n+36=18n+36. Since p(3)=0, we get 18n+36=0. Solve for n: 18n=36n=2.


Step 2: Substitute n=2 into p(x).

With n=2, the polynomial becomes p(x)=2x332x+36=2x36x+36.


Step 3: Find p(3).

Common Trap
Do not assume that because (x+3) is a factor, the answer for p(3) is 0.
p(3)=0 (because it's a factor), but p(3) is simply the value of the polynomial at x=3. Always read the question carefully!

Substitute x=3: p(3)=2(3)36(3)+36=22718+36=5418+36=72.


Therefore, p(3)=72.

So the correct answer is (D) 72.

Detailed Solution

Part (a): Mathematical Induction

Let P(n) be the proposition k=1n(1)kk2(k21)=(1)n(n1)n(n+1)(n+2)2.

Step 1: Base Case

For n=1:

L.H.S. =(1)1(1)2(121)=1(1)(0)=0
R.H.S. =(1)1(11)(1)(1+1)(1+2)2=1(0)(1)(2)(3)2=0

L.H.S. = R.H.S.

P(1) is true.

Concept Note: Always check n=1 explicitly. Note that the term (n1) in the numerator forces the result to be 0 for n=1, matching the summation logic where the term contains (k21).
Step 2: Inductive Hypothesis

Assume P(m) is true for some positive integer m. That is:

k=1m(1)kk2(k21)=(1)m(m1)m(m+1)(m+2)2 Step 3: Inductive Step

For n=m+1:

L.H.S. =k=1m+1(1)kk2(k21)
=k=1m(1)kk2(k21)+(1)m+1(m+1)2((m+1)21)

Common Trap: Don't forget to add the (m+1)-th term! The summation up to m+1 is the sum up to m PLUS the new term where k is replaced by m+1.

Using the assumption:

=(1)m(m1)m(m+1)(m+2)2+(1)m+1(m+1)2(m2+2m)

Note that (m+1)21=m2+2m=m(m+2).

=(1)m(m1)m(m+1)(m+2)2+(1)m+1(m+1)2m(m+2)

Factor out common terms. Note that (1)m+1=(1)m. We can factor out (1)mm(m+1)(m+2)2.

=(1)mm(m+1)(m+2)2[(m1)+2(1)m+1(1)m(m+1)]=(1)mm(m+1)(m+2)2[(m1)+2(1)(m+1)]=(1)mm(m+1)(m+2)2[m12m2]=(1)mm(m+1)(m+2)2[m3]=(1)mm(m+1)(m+2)2[(m+3)]=(1)m(1)m(m+1)(m+2)(m+3)2=(1)m+1((m+1)1)(m+1)((m+1)+1)((m+1)+2)2

L.H.S. = R.H.S. (where n is replaced by m+1).

P(m+1) is true.

By the principle of mathematical induction, P(n) is true for all positive integers n.

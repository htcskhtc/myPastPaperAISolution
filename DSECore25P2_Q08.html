  1. Let f(x)=x2+7x+k, where k is a constant. If f(4)+f(4)=38, find k.
    1. 6.
    2. 3.
    3. 3.

Solution We are given the function f(x)=x2+7x+k where k is a constant, and the condition f(4)+f(4)=38.
Concept Note: Remember that f(a) simply means you replace every instance of x in the equation with the number a.
First, compute f(4) and f(4) in terms of k: f(4)=42+74+k=16+28+k=44+k, Now, let's calculate f(4).
Common Trap: Be very careful when squaring negative numbers.
Correct: (4)2=16
Incorrect: 42=16 (Calculator syntax error)
f(4)=(4)2+7(4)+k=1628+k=12+k. Then substitute these back into the original sum: f(4)+f(4)=(44+k)+(12+k)=32+2k. By the given condition, 32+2k=38. Solve for k: 32+2k=382k=38322k=6k=3. Hence, k=3, so the correct option is (C).

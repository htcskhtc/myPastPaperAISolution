Solution

Part (a): Find the equation of

We are given the slope of the tangent dydx=k3x4x2. To find the equation of the curve y, we need to integrate this expression with respect to x.

y=k3x4x2dx

Split the integral into two parts to handle the constant term and the x term separately:

y=k4x2dx3x4x2dx
Concept Note: M2 Integration Techniques This split is crucial. The first part is a standard form resulting in sin1, while the second part requires integration by substitution (or inspection of the derivative of the denominator).

1. First Integral: Recall standard integral 1a2x2dx=sin1(xa)+C.

k4x2dx=ksin1(x2)

2. Second Integral: Let u=4x2, then du=2xdx or xdx=12du.

3x4x2dx=3(4x2)1/2xdx=3u1/2(12)du=32u1/21/2=34x2

Combining these results:

y=ksin1(x2)(34x2)+C y=ksin1(x2)+34x2+C

Given that passes through the origin (0,0), substitute x=0,y=0:

0=ksin1(0)+340+C 0=0+3(2)+CC=6

Part (b)(i): Range of values of k for a turning point

A turning point occurs where dydx=0.

k3x4x2=0 k3x=0x=k3
Common Trap: Domain Constraints Simply finding x=k/3 is not enough. You must ensure this x value actually lies within the domain of the function. The question states the function is defined on (2,2).

For the turning point to exist within the domain (2,2):

2<x<2 2<k3<2 6<k<6
Part (b)(ii): Point of Inflexion

To check for a point of inflexion, we examine the second derivative d2ydx2.

dydx=(k3x)(4x2)1/2

d2ydx2=(k3x)ddx[(4x2)1/2]+(4x2)1/2ddx(k3x)=(k3x)[12(4x2)3/2(2x)]+(4x2)1/2(3)=x(k3x)(4x2)3/23(4x2)1/2

d2ydx2=x(k3x)(4x2)3/23(4x2)(4x2)3/2=kx3x212+3x2(4x2)3/2=kx12(4x2)3/2

For a point of inflexion, d2ydx2=0 (and changes sign). Set the numerator to zero:

kx12=0x=12k(assuming k0)

We need to check if this solution lies within the domain (2,2). From part (b)(i), we know that for a turning point to exist, 6<k<6.

If k(6,6), then |k|<6. Consequently:

|12k|=12|k|>126=2

This means any potential solution x=12/k implies |x|>2.

Since the domain is (2,2), the value x=12/k is always outside the domain.

Pedagogical Note: Logic Check Even if d2ydx2=0 has a mathematical solution, if that x-coordinate is not on the curve (outside the interval), the feature does not exist for the function.

Conclusion: No, does not have a point of inflexion because the condition d2ydx2=0 yields no solution within the interval (2,2).

Question Code: DSE25PII_Q06
  1. If x=5.67 (correct to 2 decimal places), find the range of values of x.
    1. 5.66<x5.68.
    2. 5.66x<5.68.
    3. 5.665<x5.675.
    4. 5.665x<5.675.

Solution

Concept Note: Maximum Absolute Error When a number is rounded to a certain precision unit (e.g., nearest k), the Maximum Absolute Error is always half of that unit (k2).

Since x=5.67 is correct to 2 decimal places, it has been rounded to the nearest 0.01. Therefore, the maximum absolute error is 0.012=0.005.

This means that the actual value of x differs from 5.67 by less than 0.005. In inequality form (without using the absolute value sign), this can be written as 5.670.005<x<5.67+0.005.

Hence, 5.665<x<5.675.

Next, we consider whether the endpoints should be included.

  • When x=5.665, it rounds to 5.67 (correct to 2 decimal places), so 5.665 should be included in the range.
  • When x=5.675, it rounds to 5.68 (correct to 2 decimal places), so 5.675 should not be included in the range.
Common Trap: Boundary Inclusion Remember the "Half-up" rule used in rounding. The exact halfway point (5.665) is rounded up to 5.67, so it is included. However, the upper boundary (5.675) would round up to 5.68, so x must be strictly less than 5.675.

Mnemonic: Lower bound x< Upper bound.

Therefore, the range of possible values of x is 5.665x<5.675, which corresponds to option (D).

  1. Let a and b be constants, where 0<b<a<1. On the same rectangular coordinate system, the vertical line x=a cuts the graph of y=logax, the graph of y=logbx and the x-axis at the points A, B and C respectively. Find ABBC.
    1. 1logba
    2. 1logab
    3. logba1
    4. logab1

Visual & Step-by-Step Solution

Reference Solution Figure:

1. Determine the coordinates of points A, B, and C

The line is vertical: x=a. We substitute x=a into each equation to find the corresponding y-coordinates.

  • Point C (Intersection with x-axis): The x-axis corresponds to y=0.
    C=(a,0).
  • Point A (Intersection with y=logax):
    yA=logaa=1.
    A=(a,1).
  • Point B (Intersection with y=logbx):
    yB=logba.
    B=(a,logba).
Concept Note: Logarithm Properties Remember that logkk=1 for any valid base k. This simplifies finding Point A immediately.

2. Determine the relative vertical positions

To find the lengths AB and BC, we must know which point is higher. We are given the condition 0<b<a<1.

We need to estimate the value of yB=logba.

Using the Change of Base formula: yB=lnalnb.

  • Since 0<a<1, lna is negative.
  • Since 0<b<1, lnb is negative.
  • Therefore, the ratio is positive.
  • Since b<a, we have logba>0.
  • Also, because b<a<1, we have logba<1 (since base b<1, the inequality flips: a>blogba<logbb=1).

So, 0<logba<1.

This implies the vertical order of y-coordinates is: 0<yB<1.
Order from bottom to top: C(0)B(logba)A(1).

Common Trap: Order of Subtraction Students often incorrectly assume the order of points without checking the inequality. If you subtract in the wrong order (e.g., yCyB), you will get a negative length. Always check relative sizes: b<amagnitude of log is smaller than 1.

We require ABBC:

5. Simplify using Change of Base

Recall the property: 1logba=logab.

Substitute this back into the expression:

ABBC=logab1

Answer: This matches Option (D).

HKDSE Mathematics Practice

Question Code: DSE21PII_Q05
  1. If y=6.24 (correct to 2 decimal places), find the range of values of x.
    1. 6.23<x6.25
    2. 6.23x<6.25
    3. 6.235x<6.245
    4. 6.235<x6.245

Solution

Given y=6.24 correct to 2 decimal places, the true value x must round to 6.24 to 2 d.p.

Concept Note: Maximum Absolute Error
When a number is correct to N decimal places, the "unit" of precision is 10N. The maximum absolute error is half of this unit.

Rounding to 2 d.p. has step size 0.01, so the half-step is 0.012=0.005. Hence the boundary values are 6.240.005=6.235and6.24+0.005=6.245.

At the lower boundary, x=6.235 rounds to 6.24 (since the next digit is 5), so it is included. At the upper boundary, x=6.245 rounds to 6.25, so it is excluded.

Common Trap: Inequality Signs
Remember the standard convention for rounding ranges:
Lower Bound True Value < Upper Bound.
The Upper Bound is strictly less than (<) because if x actually equals 6.245, it would round up to 6.25.

Therefore, 6.235x<6.245 which corresponds to option C.

Consolidated Notes & Strategy

Notes
  • Maximum Absolute Error: For a measurement correct to k decimal places, the max absolute error is 1210k.
  • Range Notation: The range of true values lies between:
    Measured Value Error x< Measured Value + Error.
Strategy
  1. Identify the precision level (e.g., 2 decimal places means units of 0.01).
  2. Calculate the error tolerance (Half of the unit: 0.012=0.005).
  3. Subtract this from the given value to find the Lower Bound (Inclusive ).
  4. Add this to the given value to find the Upper Bound (Exclusive <).
Required Skill
  • Understanding of rounding mechanisms (significant figures vs. decimal places).
  • Handling compound inequalities for error intervals.

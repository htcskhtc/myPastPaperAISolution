DSE22PII_Q06

6. It is given that x is a real number. If x is rounded down to 3 significant figures, then the result is 345. Find the range of values of x.

  1. 344<x345
  2. 345x<346
  3. 345<x345.5
  4. 344.5x<345.5

Solution

To round down to 3 significant figures means to keep the first 3 significant digits and obtain the largest 3-s.f. number that is x.

Concept Note: Rounding Methods It is crucial to distinguish between Rounding Off (nearest) and Rounding Down (truncate).
  • Round Off: Looks at the next digit to decide whether to round up or keep. Range is usually 0.5place value.
  • Round Down: Simply cuts off digits after the specified place. This creates an interval starting from the value itself up to the next increment.

Since the rounded-down result is 345 (which already has 3 significant figures):

  • The smallest value of x that rounds down to 345 is x=345 (because 345 rounded down is 345).
  • The next 3-s.f. number after 345 is 346. If x346, rounding down to 3 significant figures would give 346 (or higher), not 345.

Hence,

345x<346.

Therefore, the correct option is B.

Common Trap Many students confuse this with "rounding off to the nearest integer" or "nearest 3 s.f.".
If the question said "rounded off to 3 s.f.", the answer would be 344.5x<345.5 (Option D).
Always check if the keywords are "rounded down", "rounded up", or "rounded off".

Consolidated Notes & Strategy

Notes Definitions of Approximation Methods:
Let N be the place value of the last significant digit kept (the "unit").

1. Rounding Down (Truncate): The value x satisfies:
Resultx<Result+N 2. Rounding Up (Ceiling): The value x satisfies:
ResultN<xResult 3. Rounding Off (Nearest): The value x satisfies:
Result0.5Nx<Result+0.5N
Strategy 1. Identify the precision unit: In this question, 345 has 3 s.f. The last digit '5' is in the units place (100). So the step size (unit) is 1.
2. Apply the specific rule: Since it is "rounded down", the range starts at the number itself and extends 1 unit upwards (exclusive).
3. Check endpoints: Test the boundary numbers explicitly. Does 346 round down to 345? No, it becomes 346. Thus x<346.
